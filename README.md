## Sobre o Projeto

Portfólio pessoal desenvolvido em **React + Vite**, com foco em design moderno e sofisticado. O projeto apresenta minha trajetória profissional, habilidades técnicas e projetos desenvolvidos, com animações suaves e layout totalmente responsivo.

---

## Seções

| Seção | Descrição |
|---|---|
| **Hero** | Apresentação com animações e links sociais |
| **Sobre** | Resumo profissional e estatísticas de carreira |
| **Skills** | Barras de progresso animadas por tecnologia |
| **Experiência** | Timeline com histórico profissional |
| **Projetos** | Cards com projetos desenvolvidos |
| **Contato** | Links de contato direto |

---

## Tecnologias

- **React 19** — biblioteca de interface
- **Vite 8** — bundler e dev server
- **Framer Motion** — animações declarativas
- **React Scroll** — scroll suave entre seções
- **React Icons** — biblioteca de ícones
- **CSS Modules** — estilização com escopo isolado

---

## Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/GustavoACarriel/portfolio

# Entre na pasta
cd portfolio

# Instale as dependências
npm install

# Rode em desenvolvimento
npm run dev
```

Acesse em `http://localhost:5173`

---

## Build para produção

```bash
npm run build
```

O resultado estará na pasta `dist/`.
